# 🤖 Basic Chatbot - CodeAlpha Task 4
## 📌 Overview

This project is a simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.
The chatbot interacts with users through the console by responding to predefined inputs such as greetings, common questions,
and exit commands. It demonstrates the use of Python fundamentals to create an interactive command-line application.
---

## 🚀 Features
- Interactive console-based chatbot
- Predefined responses for common user queries
- User-friendly conversation flow
- Handles unknown inputs gracefully
- Exit the chatbot using the "bye" command
- Easy to understand and modify
---

